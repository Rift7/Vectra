# Vectra

Vectra is a slicer-style web app for preparing artwork for pen plotters.

## Repo Layout
- `apps/web`: Next.js frontend
- `apps/api`: FastAPI backend
- `packages/shared`: Shared schemas and types

## MVP Goals
- Import SVG/PNG/JPG
- Vectorize raster via vtracer
- Process vectors via vpype
- Optimize toolpaths and export G-code
- Live preview playback
